# Vision Doctor - AI 기반 소(젖소) 렌즈 오염 탐지 시스템

이 프로젝트는 소 농장 CCTV 카메라 렌즈에 발생하는 오염(먼지, 물방울 등)을 실시간(시뮬레이션)으로 감지하고, AI 진단 리포트를 제공하는 통합 관제 UI를 구현합니다.

## 주요 기능
- **YOLO11 모델**을 이용한 고정밀 오염 객체 탐지
- **CustomTkinter** 기반 다크 테마 대시보드
- **시뮬레이션 모드**: 실제 하드웨어 없이 랜덤 이미지 스캔 및 가상 하드웨어 상태 표시
- **AI 진단 리포트**: 오염 감지 시 자동 텍스트 출력 및 와이퍼 동작 시뮬레이션 로그
- **학습 지표 차트**: mAP 및 손실 그래프 표시

## 실행 방법
1. 의존성 설치: `pip install -r requirements.txt`
2. 애플리케이션 실행: `python vision_doctor_system.py`

## 데이터셋
- 1,000장 이상의 소와 렌즈 오염 이미지 포함
- `bg_` 혹은 손바닥·인간 이미지 등 개인정보가 포함된 파일은 모두 제외
- 모든 이미지와 모델 파일은 Git LFS 로 관리됩니다.

## 주의사항
이 데이터셋은 연구·학습용으로만 제공되며, 개인 식별 정보(손바닥, 얼굴 등)는 포함되지 않음.
